# FoodEase

FoodEase is a **fully dynamic web application** designed to make food ordering and browsing seamless, fun, and user-friendly. Users can explore restaurants, view menus, add items to the cart, and enjoy a modern, responsive UI. The website works both **online and offline** for better accessibility.
## Features

- **Dynamic Restaurant and Menu Browsing:** Fetch and view all restaurants and menu items in real-time.  
- **Top Restaurants at a Click:** Quickly view the most popular or top-rated restaurants.  
- **Quick Search:** Easily search for your favorite food or dish.  
- **Cart Management:** Add, remove, and view cart items dynamically.  
- **Cool UI:** Modern, responsive, and interactive interface using Tailwind CSS.  
- **Offline/Online Support:** Users can browse the website offline (with limited functionality).  
- **Detailed Item Overview:** See prices, ratings, delivery time, and item details with a click.  
- **Interactive Elements:** Smooth animations, hover effects, and interactive buttons for better UX.

  ## Technologies Used

- **Frontend:** React.js, HTML, CSS, Tailwind CSS, JavaScript    
- **State Management:** Redux / Redux Toolkit  
- **Version Control:** Git & GitHub
